# Digestive System Image Sources

The Grade 8 digestive-system cached images in this directory are derived from these open medical illustration sources:

- `Blausen 0316 DigestiveSystem.png` by BruceBlaus / Blausen.com staff, licensed under CC BY 3.0: https://commons.wikimedia.org/wiki/File:Blausen_0316_DigestiveSystem.png
- `Human Digestive Tract (NIH BioArt 212 - 657931).png`, courtesy of NIAID / Ryan Kissinger, public domain as United States government material: https://commons.wikimedia.org/wiki/File:Human_Digestive_Tract_(NIH_BioArt_212_-_657931).png

Changes made for the application include cropping, removal of source labels and leader lines, resizing, compositing into 16:9 slide-safe canvases, and adding non-text instructional overlays such as arrows, focus rings, process symbols, and villi/waste path panels.
